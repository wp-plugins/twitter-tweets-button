Twitter button plus is nice and useful tool to show Twitter tweet button on your website. 

== Description ==
Twitter button plus is awesome tool to show your visitors your popularity on Twitter. At the same time it's very useful, because you will get new users who will follow your posts. Twitter button plus is important part of social optimization, that's why a lot of people use Twitter button plus on their websites.
It's very easy to use, you just need to install and configure Twitter button plus options.

### Features of Twitter button plus:

* **User friendly and easy to use.**
* **Tested with other popular plugins.**
* **Ability to change Twitter Button language.**
* **Ability to set Twitter Account.**
* **Ability to set Related Twitter Account.**
* **Ability to choose tweet text.**
* **Ability to choose where to display tweet button(pages, front page).**
* **Ability to display Twitter Button before content.**

### Features of [Twitter button plus Pro](http://wpdevart.com/wordpress-twitter-plugin/):

* **Ability to Choose Twitter Button Style.**
* **Ability to display Twitter Button after content.**
* **Ability to type your own tweet text.**

== Installation ==

### That's simple:

1. Download Twitter button plus plugin.   
2. Upload the Twitter button plus plugin from your admin panel.    
3. Activate the plugin. 
4. Then configure the Twitter button plus options.
